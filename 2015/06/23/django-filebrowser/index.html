django1.8 服务器目录管理 -- grappelli + filebrowser

发表于 2015-06-23   |  

FileBrowser是对Django 管理员界面的一个扩展，目的是：

  • 浏览服务器上的目录，包括上传、删除、编辑和重命名文件。
  • 使用FileBrowseField将图像/文档添加到django的models或database中。
  • 使用TinyMCE选择图像或文档。

安装要求：

  • Django1.8
  • Grappelli

    pip install django-grappelli

  • Pillow

    Pillow是PIL的替代版本，，如：改变图像大小，旋转图像，图像格式转换，色场空间转换，图像增强，直方图处理，插值和滤波等等。
    首先安装一些图像处理包：
    apt-get install libjpeg-dev
    apt-get install libfreetype6-dev
    apt-get install zlib1g-dev
    apt-get install libpng12-dev
    然后安装Pillow
    #pip install pillow
